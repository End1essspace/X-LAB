SCapturer v{{VERSION}}
====================

Performance-first Windows screenshot developer utility.
Created by XCON as part of X-LAB.

RUN
---
1. Keep SCapturer.exe in a stable folder.
2. Launch SCapturer.exe to open the management console.
3. Closing the console with X returns the listener to the background.
4. Launch SCapturer.exe again or press Ctrl+Shift+H to show the console.

DEFAULT HOTKEYS
---------------
Ctrl+Shift+G  Capture the complete virtual desktop
Ctrl+Shift+S  Capture a selected rectangular region
Ctrl+Shift+H  Show or hide the management console
Ctrl+Shift+Q  Exit SCapturer gracefully

DATA
----
Settings and diagnostics:
%LOCALAPPDATA%\SCapturer

Screenshots:
%USERPROFILE%\Pictures\SCapturer\Full
%USERPROFILE%\Pictures\SCapturer\Snips

PORTABLE REMOVAL
----------------
Exit SCapturer with Ctrl+Shift+Q or SCapturer.exe --exit, then delete the
portable folder. User screenshots, settings, and diagnostics are retained.
